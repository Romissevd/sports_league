Про сайт

Этот сайт разрабатывается для автоматизации процесса обработки ставок сделанных зарегистрированными участниками различных турниров

Данные матчей будут браться с использованием свободных API с различных ресурсов

Участнику до начала матча (поединка, раунда и т.п.) предоставляется возможность выполнить ставку во время матча и после сделать ставку нельзя.

Происходит подсчет результатов и вытсраивается своеобразная турнирная таблица среди участников
